संस्कृते पदम् नाम भागद्वयं स्त:
	
पदम् = प्रकृति: + प्रत्यय:

```
पदम् द्विविधम् 
	तिङन्तम् (क्रियापदम्) = धातु: (प्रकृति:) + तिङ् प्रत्यय:
	सुबन्तम् (नामपदम्) = प्रातिपदिकम् (प्रकृति:) + सुप् (प्रत्यय:)
```
